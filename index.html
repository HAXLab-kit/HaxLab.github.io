Research

Multimodal Interaction

We explore multimodal interaction techniques for collaborative environments, combining modalities such as gaze, speech, and touch.

  • Real-time multi-user collaboration in simulated manufacturing environments
  • Tabletop touchpad technique enabling finger-based interaction combined with gaze input
  • Overcoming conventional pinch gesture limitations

Immersive Interaction

We investigate immersive VR experiences and accessible interaction for diverse users.

  • 2D image processing into cubemap format for fully immersive 3D road view VR
  • Gesture-based controls for digitally marginalized users
  • Investigation of virtual grasping methods across diverse object shapes
  • Analysis of objective and subjective measures

Collaborative Interaction

We develop interaction techniques that enable users to collaborate easily and intuitively in real-time multi-user virtual environments for assembly simulations.

  • Combining gesture, gaze, and speech modalities
  • Shared virtual space design for multi-user assembly

Embodied XR Interaction

We estimate full-body motion in VR using only head movement data, eliminating the need for external sensors.

  • Upper and lower body motion estimation using only head movement data
  • Hardware-free approach eliminating external sensors
  • Real-time avatar interaction for multi-user VR
